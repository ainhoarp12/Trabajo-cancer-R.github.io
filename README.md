# Trabajo-cancer-R.github.io
Somos María Pérez Luna, Ainhoa Romero Pérez y Andrea Ponce Pomares. Para nuestro proyecto de fin de curso de la asignatura de Computación e Interpretación de datos en Biologia, hemos elegido analizar una base de datos sobre la Genómica de la sensibilidad a los medicamnetos en el cáncer (GDSC). Nos hemos centrado en hacer una comparativa de sensibilidad del fármaco Camptothecin entre cáncer de pulmón y mama. Este análisis puede resultar de gran utilidad para conocer qué tan efectivo es el fármaco y para cuál de los dos tipos de cáncer es más adecuada su administración. A partir de esto, se puede contribuir al desarrollo de futuras terapias más específicas y efectivas.
Para poder visualizar este archivo, debe tener instaladas las siguientes librerías: library(dplyr), (ggplot2), (readxl) 
Además, cabe destacar que, del archivo principal de datos, hemos hecho una subselección en la que se encuentran unicamente los datos con los que vamos a trabajar.
